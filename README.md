# Voting Bot

Telegram-бот на Ruby для создания опросов на основе локальной библиотеки `VotingWizard`.

## Что умеет бот

- создаёт опросы через пошаговый сценарий со состояниями;
- сохраняет состояния пользователей и опросы в JSON, поэтому переживает рестарт;
- использует объектные обработчики состояний вместо одного большого `case`;
- позволяет просматривать список опросов, голосовать и смотреть результаты.

## Команды

- `/start` - приветствие и краткая инструкция;
- `/help` - список команд;
- `/new_poll` - начать создание опроса;
- `/polls` - показать все доступные опросы;
- `/my_polls` - показать опросы, созданные вами;
- `/vote ID` - начать голосование в опросе;
- `/results ID` - показать результаты опроса;
- `/cancel` - отменить текущий сценарий.

## Запуск

1. Установите зависимости:

```bash
bundle install
```

2. Укажите токен Telegram-бота:

```bash
$env:TELEGRAM_BOT_TOKEN="your_token"
```

3. Запустите бота:

```bash
bundle exec ruby bin/start_bot
```

По умолчанию бот хранит данные в `db/bot_state.json`. Путь можно переопределить через `BOT_STORAGE_PATH`.
